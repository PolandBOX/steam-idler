![](screenshot.png)

## INstalacja

1. Pobierz repozytorium

```bash
git clone https://github.com/PolandBOX/steam-idler.git
```
2. Zainstaluj moduły

```bash
npm install
```
3. Zmień nazwę configu i go wypełnij.

4. Uruchom program

```bash
npm start
```
